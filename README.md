# Ashmita Dutta — Personal Portfolio

A responsive personal portfolio website built with vanilla HTML, CSS, and JavaScript.

## Sections

- **Home** — Typed role animation with hero intro
- **About** — Bio, personal details, and links
- **Education** — Timeline of academic background (KIIT, ISC, ICSE)
- **Experience** — Software Development Trainee & SIH participation
- **Skills** — Animated skill bars across Languages, Web Dev, and Tools
- **Projects** — Student Record Management System, Portfolio Website, ML Project
- **Contact** — Form with client-side validation

## Tech Stack

| Layer | Tech |
|-------|------|
| Markup | HTML5 |
| Styling | CSS3 (custom properties, grid, flexbox) |
| Scripting | Vanilla JavaScript |
| Fonts | Google Fonts — Playfair Display, Inter |
| Icons | Devicons (CDN) |

## Features

- Fixed navbar with scroll shadow effect
- Typed text animation cycling through roles
- Intersection Observer–based scroll reveal animations
- Animated skill progress bars triggered on scroll
- Responsive hamburger menu for mobile
- Contact form with email format validation

## Project Structure

```
Portfolio/
├── index.html       # Main HTML file
├── style.css        # All styles and responsive layout
├── script.js        # Animations, interactions, form validation
├── mefornal.jpg     # Hero section photo
├── mycasual.jpeg    # About section photo
└── sihpic.jpeg      # SIH experience photo
```

## Getting Started

Just open `Portfolio/index.html` in any modern browser — no build step or dependencies required.
